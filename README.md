# Kanini

## Abstruct

C言語からRustへのトランスパイラを作成する
最終的な目標はC言語からRustへ人間の読めるコードで変換すること

## System

main.rs -> calc.rsからparser呼び出し
parser.rsでパース処理
パース処理過程でast.rsのastの内容が並行して実行される
