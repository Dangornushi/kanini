# Kanini

## Abstruct

C 言語から Rust へのトランスパイラを作成する
最終的な目標は C 言語から Rust へ人間の読めるコードで変換すること

## System

main.rs -> calc.rs から parser 呼び出し
parser.rs でパース処理
パース処理過程で ast.rs の ast の内容が並行して実行される
